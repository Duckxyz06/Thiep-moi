HƯỚNG DẪN SETUP RSVP
1. Tạo Google Sheet → hàng 1: Timestamp | Tên | Tham dự | Thiệp
2. Extensions → Apps Script → dán code:
function doPost(e){
  var s=SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  var d=JSON.parse(e.postData.contents);
  s.appendRow([new Date()
    .toLocaleString('vi-VN'),
    d.ten,d.thamDu,d.thiep]);
  return ContentService
    .createTextOutput('ok');
}
3. Deploy → Web app → Anyone → Copy URL → dán vào tab Sự kiện
